## 1. Hamburger 클래스 만들기

---

**kr.hs.sdh.workbook2.entity** 패키지에 **Hamburger** 클래스를 아래의 JSON 데이터 형식에 맞게 작성하시오.

```json
[
  {
    // 햄버거 이름
    "name": "통새우크런KIM버거",
    // 햄버거 가격
    "price": 7700,
    // 햄버거 이미지 경로
    "imagePath": "/images/통새우크런KIM버거.png",
    // 신제품 여부
    "isNew": false,
    // 추천 여부
    "isRecommended": false
  },
  { ... },
]
```
